Данный транслятор переводит программу, написанную на встроенном языке, на язык машинных команд, затем компилирует её в *.exe файл.

#Работа с интерфейсом

Исходный код вводится в текстовое поле слева. Перед выполнением компиляции код нужно сохранить.
После компиляции, если в процессе анализа не было выявлено ошибок, в правом текстовом поле появится созданный код на языке Ассемблера, и будет выполнена компиляция этого кода в .exe файл.
В случае, если в коде были найдены ошибки, они будут выведены в текстовое поле снизу, с указанием строки, в которой возникла ошибка.

##Команды меню:
* Файл
  * Новый - очищает текстовые поля.
  * Открыть - открывает файл и заполняет текстовое поле кодом из этого файла.
  * Сохранить - сохраняет код в открытый ранее файл, или, если код новый, запускает диалоговое окно сохранения файла.
  * Сохранить как... - запускает диалоговое окно сохранения файла и сохраняет код в выбранном пути.
  * Выход - закрывает окно приложения.
* Компиляция
  * Выполнить - выполняет трансляцию и компиляцию исходного кода.

#Необходимые файлы:
ml.exe - компилятор MASM32
link.exe - компоновщик
kernel32.lib - библиотека системных функций
mspdb50.dll - библиотека компоновщика

Все вышеперечисленные файлы необходимо поместить в директорию, содержащую основной исполняемый файл проекта. Стандартный путь для этого файла - TranslatorUI/bin/Debug или TranslatorUI/bin/Release

#Описание языка

## Возможности

Язык позволяет проводить арифметические операции с переменными заданного типа. Поддерживаются четыре основные операции и выражения в скобках.
Есть логический оператор If для проведения разветвлённых вычислений, а также оператор перебора вариантов Case.

##Грамматика языка

<Программа> ::= <Объявление переменных> <Описание вычислений> <Оператор печати>
<Описание вычислений> ::= Begin <Список инструкций> End
<Объявление переменных> ::= Var <Список переменных> 
<Список переменных> ::= <Идент> | <Идент> , <Список переменных> 
<Список инструкций>::= <Инструкция> | <Инструкция> <Список инструкций>
<Инструкция> ::= <Идент> = <Выражение> | <Условный оператор> | <Перебор вариантов> 
<Выражение> ::= ( <Выражение> ) | <Операнд> | <Выражение> <Бин.оп.> <Выражение>
<Условный оператор> ::= If <Условие> <Окончание условия>
<Условие> ::= <Логическое выражение> Then <Список инструкций>
<Логическое выражение> ::= <Выражение> <Лог. оп.> <Выражение>
<Окончание условия> ::= EndIf | ElseIf <Условие> <Окончание условия> | Else <Список инструкций> EndIf
<Перебор вариантов> ::= Case <Идент> Of <Список вариантов> EndCase
<Список вариантов> ::= <Вариант> <Список вариантов> | <Вариант>
<Вариант> ::= <Const> : <Список инструкций>
<Бин.оп.> ::= "-" | "+" | "*" | "/"
<Лог. оп.> ::= "!=" | "==" | ">" | ">=" | "<" | "<="
<Операнд> ::= <Идент> | <Const> 
<Идент> ::= <Буква> <Идент> | <Буква>
<Const> ::= <Цифра> <Const> | <Цифра> 
<Оператор печати>::=Print <Идент>
